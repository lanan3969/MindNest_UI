# 手势教学资源文件夹

本文件夹用于存放6种手势的教学动画和说明资源。

## 文件结构

每个手势应有一个子文件夹，包含以下内容：

```
GestureTutorials/
├── stroke_tutorial/
│   ├── frame_0.png
│   ├── frame_1.png
│   ├── frame_2.png
│   └── description.txt
├── poke_tutorial/
├── feed_tutorial/
├── hug_tutorial/
├── wave_tutorial/
└── heart_tutorial/
```

## 动画帧

- 格式：PNG
- 分辨率：建议300x300像素
- 数量：每个手势3-10帧
- 命名：`frame_0.png`, `frame_1.png`, `frame_2.png`...

## 说明文本

每个手势的`description.txt`包含分步骤说明。

当前说明已内置在`GestureTutorialUI.cs`中，但可以通过此文件覆盖。

## 当前状态

**注意**：教学动画是可选的！

- 如果没有动画帧，系统会自动显示手势图标（来自`GestureIcons/`文件夹）
- 系统已内置文字说明，无需额外配置
- 用户第一次遇到某个手势时会自动显示教学

## 添加动画的方法

### 方法1：使用序列帧图片

1. 录制或绘制手势演示
2. 导出为序列帧PNG
3. 放入对应的tutorial文件夹
4. 系统会自动加载并循环播放

### 方法2：使用GIF

1. 准备GIF动画
2. 在Unity中导入
3. 在`GestureTutorialUI.cs`中修改加载逻辑

### 方法3：仅使用文字

不需要任何额外文件！系统已经内置了所有手势的文字说明。

## 当前教学内容

### 抚摸 (Stroke)
1. 将手掌展开，五指自然分开
2. 手掌平行于屏幕
3. 在Nomi附近缓慢水平移动
4. 保持动作1-2秒

### 戳戳 (Poke)
1. 伸出食指，其他手指弯曲
2. 快速将食指指向Nomi
3. 快速回退手指
4. 整个动作要迅速完成

### 投喂 (Feed)
1. 手掌向上，像捧着东西
2. 手从Nomi下方开始
3. 缓慢向上移动到Nomi位置
4. 保持手掌向上的姿势

### 抱抱 (Hug)
1. 伸出双手，手掌朝向Nomi
2. 双手分别在Nomi左右两侧
3. 同时向Nomi靠近
4. 像给Nomi一个拥抱

### 挥手 (Wave)
1. 抬起手，手掌展开
2. 在Nomi附近左右摆动
3. 摆动3次以上
4. 动作要清晰明显

### 比心 (Heart)
1. 伸出双手到Nomi上方
2. 拇指和食指靠近
3. 两只手形成心形
4. 保持姿势1-2秒

---

**提示**：即使不添加任何动画资源，教学系统也能正常工作！
